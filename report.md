# Отчет по проекту AnswerHub

Авторы: virsi
Репозиторий: virsi/AnswerHub
Дата: 2025-12-23

## 1. Краткое описание проекта
AnswerHub — проект для VK TechPark, цель которого — разработка веб-приложения для хранения, поиска и управления ответами/записями знаний (Knowledge Base). Проект сочетает фронтенд на HTML/CSS/JavaScript и бэкенд на Python.

## 2. Цели и задачи
- Создать удобный интерфейс для добавления, редактирования и поиска записей.
- Обеспечить хранение данных и API для взаимодействия фронтенда и бэкенда.
- Реализовать поиск по ключевым словам и фильтры.
- Подготовить проект для демонстрации руководителю и дальнейшей доработки.

## 3. Технологический стек
- Фронтенд: HTML, CSS, JavaScript (примерно 44.5% HTML, 14% CSS, 7.6% JS в репозитории).
- Бэкенд: Python (33.4%).
- СУБД: (уточнить — SQLite/PostgreSQL/другое). Текущая реализация использует файл/локальную БД (необходимо проверить по коду).

## 4. Архитектура и структура репозитория
- Веб-страницы и статические файлы: директории с HTML/CSS/JS.
- Серверная часть: файлы на Python, реализующие API и логику работы с данными.
- Файловая структура (основные папки и файлы, пример):
  - templates/ или static/ — фронтенд-шаблоны и ресурсы
  - app.py / main.py / server.py — точка запуска бэкенда
  - requirements.txt — зависимости (если есть)
  - README.md — инструкция по запуску (если есть)

## 6. Результаты и демонстрация
- Какие фичи уже работают: добавление записи, просмотр записи, поиск по ключевым словам, фильтрация по категориям (необходимо уточнить список рабочих функций по коду).
- Скриншоты/видео для презентации: рекомендуется подготовить 3–5 слайдов с ключевыми экранами (главная, форма добавления, результат поиска, пример записи).

## 7. Как запустить проект локально (примерные шаги)
1. Клонировать репозиторий: `git clone https://github.com/virsi/AnswerHub.git`
2. Перейти в папку проекта: `cd AnswerHub`
3. Создать и активировать виртуальное окружение: `python -m venv venv` / `source venv/bin/activate`
4. Установить зависимости: `pip install -r requirements.txt` (если файл есть)
5. Запустить сервер: `python app.py` или `flask run` (уточнить по коду)
6. Открыть в браузере: `http://127.0.0.1:5000`
